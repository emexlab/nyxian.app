# Installation

Remember Nyxian is mainly developed for jailed devices, so it's the recommended way of installing Nyxian.

<Tabs>
   <TabItem value="jailed" label="Jailed" default>
      ## Jailed

      :::info[Important]

      Do not forget to keep all extensions when installing!
      
      :::

      :::warning

      Nyxian will only work, if sideloaded with an apple developer account (free or paid)  
      **It will not work in LiveContainer!**

      :::

      You will need a free or paid apple developer account, which you have to use to sign and install Nyxian.
      Your developer certificate will be required again in the ["Importing your Developer Certificate"](#importing-your-developer-certificate) section.

      ### Requirements

      - iOS/iPadOS 16+
      - AltStore 2.0+ / SideStore 0.6.0+

      :::note

      You can find a guide for installing SideStore [here](https://docs.sidestore.io/docs/installation/prerequisites)

      :::

      ### Download
      <a href="https://stikstore.app/altdirect/?url=https://nyxian.app/apps.json&exclude=livecontainer" target="_blank">
         <img src="https://github.com/stikstore/altdirect/blob/main/assets/png/AltSource_Blue.png?raw=true" alt="Add AltSource" width="200" />
      </a>
      <a href="https://github.com/ProjectNyxian/Nyxian/releases/download/0.9/nyxian.ipa" target="_blank">
         <img src="https://github.com/stikstore/altdirect/blob/main/assets/png/Download_Blue.png?raw=true" alt="Download .ipa" width="200" />
      </a>

      ### Importing your Developer Certificate
      Open the store you used to install Nyxian and goto it's settings tab. At the bottom there should be an `Export Signing Certificate...` button. Click it and enter any password (you will just need to remember it for a short time).  
      Now open Nyxian and in the settings tab select `Certificate`. Now just click `Import Certificate`, pick your previously exported certificate and enter its password.

      If everything went well you should see <span style={{ color: 'green' }}>`Certificate Valid`</span>.
