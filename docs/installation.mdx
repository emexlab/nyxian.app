# Installation

## Jailed
:::info
Do not forget to keep all extensions when installing!
:::

Jailed is the main version of Nyxian. You need a free or paid apple developer account, which you have to use to sign and install Nyxian and in Nyxian you have to import the same developer certificate.
**Do not use LiveContainer** to use Nyxian, install Nyxian seperately.

### Requirements

- iOS/iPadOS 16+
- AltStore 2.0+ / SideStore 0.6.0+

### Importing your Developer Certificate
Open the store you used to install Nyxian and goto it's settings tab. At the bottom there should be an `Export Signing Certificate...` button. Click it and enter any password (you will just need to remember it for a short time). Then open Nyxian and select the settings tab. In there select `Certificate` and then `Import Certificate`. Now just pick your previously exported certificate and enter the password.

If everything went well you should see <span style={{ color: 'green' }}>`Certificate Valid`</span>.
